DJ SKILL
  • Transforms robotic agent announcements into creative, human-sounding radio callouts with personality and dry humor
  • Install: download dj.skill, then run claude skill install dj.skill or unzip into ~/.claude/skills/dj/
  • Tone-aware: adapts voice for completions (satisfied), failures (calm alert), stuck (patient), and milestones (momentous)
  • Works with any TTS voice. Keeps announcements concise, never pads short messages, never uses exclamation points
AMBIENT TONES
Subtle sound effects that play under the music, giving you a sense of what agents are doing without words.
  • Rising chime: agent started work
  • Warm chord: work completed
  • Tense note: something failed
  • Low pulse: agent is stuck
  • Soft hum: agent idle
  • Falling notes: agent stopped
  • Bright bell: deploy event
  • Long chord: milestone reached
Tones that signal problems (failed, stuck) also play a voice announcement. Start, stop, and idle are tone-only.
